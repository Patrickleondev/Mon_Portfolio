# AI RedTeaming Portfolio & Blog

Bienvenue sur le dépôt de mon portfolio professionnel orienté **AI RedTeaming** et **CyberSécurité**.

##  Architecture

Ce projet utilise une architecture hybride :
- **Landing Page** : Construite avec [Vite](https://vitejs.dev/) (HTML/JS/CSS natif) pour une expérience "Hacker" immersive (Matrix effect, Terminal style).
- **Blog** : Géré par [Jekyll](https://jekyllrb.com/) sous le répertoire `/blog` pour les writeups CTF et rapports de labs.

##  Stack Technique

- **Frontend** : Vanilla JS, CSS3 (Custom animations), Fira Code Font.
- **Blog** : Jekyll, Minima Theme (Customized).
- **CI/CD** : GitHub Actions (Déploiement automatique sur GitHub Pages).
- **Conteneurisation** : Docker (Multi-stage build).

##  Structure du projet

- `index.html` : Page d'accueil interactive.
- `blog/` : Source du blog Jekyll.
- `.github/workflows/` : Pipeline de déploiement.
- `Dockerfile` : Configuration du build hybride Node/Jekyll.
